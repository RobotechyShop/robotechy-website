:toc:
:toc-placement: preamble

This document defines the visual identity and design standards for the Robotechy website and brand.

== Brand Overview

Robotechy is a 3D printing store specializing in Bitcoin Seed Signer cases and accessories. The brand identity reflects a modern, tech-forward aesthetic with a focus on Bitcoin and maker culture.

=== Colors to Avoid

* *No blue tints* - The brand deliberately avoids blue. All greys should be neutral or have a subtle green tint.
* Slate colors from Tailwind (e.g., `slate-400`, `slate-600`) have a blue tint and should not be used for text.

== Typography

=== Price Display

Prices are a key element and use the brand green:

* *Light mode*: `text-robotechy-green-dark` (#5a9106) for readability
* *Dark mode*: `text-robotechy-green` (#9efe09) for vibrancy

=== Text Hierarchy

* *Headings*: Bold, `text-slate-900 dark:text-white`
* *Body text*: `text-sage-600 dark:text-sage-400`
* *Muted/secondary*: `text-sage-500 dark:text-sage-400`
* *Links/interactive*: `text-robotechy-green-dark` with hover state

== Dark Mode

=== Implementation

Dark mode is implemented using Tailwind's class-based approach (`darkMode: "class"`).

=== Theme Toggle

The header includes a theme toggle that cycles through:

1. *Light* (Sun icon) - Light backgrounds, dark text
2. *Dark* (Moon icon) - Dark backgrounds, light text
3. *System* (Monitor icon) - Follows OS preference

== Design Principles

1. *No blue* - All greys are neutral or green-tinted
2. *High contrast* - Prices and CTAs should stand out
3. *Consistency* - Use shared components (e.g., `<Header />`) across all pages
4. *Adaptive* - Colors should work well in both light and dark modes
5. *Bitcoin-forward* - The aesthetic should appeal to the Bitcoin community
